# Dataset

Source:
120 Years of Olympic History (Coursera)

Files:
- athlete_events.csv
- noc_regions.csv

These datasets are used for the analyses presented in this project. 

> ⚠️ **Note on Data Ingestion:** Due to its large size, the `athlete_events.csv` file could not be imported directly into the repository. It has been processed via an external data pipeline (such as Databricks / dbt) or needs to be downloaded locally before running the project.
