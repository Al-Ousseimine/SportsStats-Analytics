## 📌 Project Overview

SportsStats is a sports analytics company that collaborates with journalists and elite coaches to transform historical Olympic data into actionable insights.

This project analyzes more than **120 years of Olympic Games history**, following a complete analytics workflow from raw data preparation to interactive business dashboards.

The objective is to uncover meaningful trends in athlete participation, country performance, medal distribution and Olympic evolution through SQL analysis and Power BI visualizations.

## 🎯 Business Problem

SportsStats collaborates with journalists and sports professionals to transform Olympic data into meaningful insights.

The company needs a reliable analytical solution capable of answering strategic questions such as:

- Which countries dominate Olympic history?
- How has Olympic participation evolved?
- Which nations are the most efficient?
- How are medals distributed across Olympic history?

The goal of this project is to transform raw Olympic data into actionable business insights through SQL analysis and interactive Power BI dashboards.

## 📂 Dataset

The dataset contains more than 120 years of Olympic Games history.

Main entities include:

- Athletes
- Countries
- Olympic Games
- Sports
- Events
- Medals

Main tables:

- athlete_events.csv
- noc_regions.cs

## 🏗️ Solution Architecture

The project follows a complete analytics pipeline.

Raw CSV files are imported into Databricks where the data is cleaned and transformed.

Analytical SQL views are then created to support business analysis.

Finally, Power BI connects to these analytical datasets to build interactive dashboards used by business stakeholders.
