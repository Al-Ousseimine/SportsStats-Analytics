# SportsStats-Analytics

Projet d'analyse et de visualisation de données sportives.

## Structure du projet

- **data/** : Données brutes (`raw`) et traitées (`processed`).
- **notebooks/** : Notebooks Jupyter / Python pour l'exploration.
- **sql/** : Scripts et requêtes SQL.
- **powerbi/** : Rapports et fichiers de conception Power BI.
- **databricks/** : Notebooks et pipelines Databricks.
- **documentation/** : Notes, rapports et architecture.
- **images/** : Captures d'écran et graphiques pour illustrer le projet.
